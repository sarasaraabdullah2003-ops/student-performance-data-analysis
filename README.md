# Student Performance Data Analysis

This project analyzes student performance data using Python.

## Tools Used

- Pandas
- NumPy
- Matplotlib
- Seaborn
- Plotly
- Scikit-learn
- Statsmodels

## Project Goals

- Analyze student grades
- Study the relationship between study hours and final scores
- Visualize the data
- Build a simple prediction model

## Dataset

The dataset contains information about 500 students including:

- Study hours
- Attendance
- Subject scores
- Final grade

## Results

- Study hours have a positive effect on student performance
- Students with higher attendance tend to achieve better scores# student-performance-data-analysis
